1. **首页** - 个人介绍与数据概览
2. **关于我** - 专业背景与核心能力

## 🚀 如何访问

### 方法1：直接打开（推荐）
1. 双击 `launch.html`
2. 点击"立即打开简历网站"按钮
3. 或直接双击 `index.html`

### 方法2：分享给他人
1. 发送整个 `game-ua-resume` 文件夹
2. 对方双击 `index.html` 即可打开
3. 无需安装任何软件

### 方法3：部署到服务器
```bash
# 使用Python启动本地服务器
cd game-ua-resume
python3 -m http.server 8000

# 访问 http://localhost:8000
```
